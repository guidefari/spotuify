Notes:
- Single-row layout at ≥144 cols (or when ≤3 groups visible);
  two-row grid when there are >3 groups and width <144.
- Each card uses `card_block` (ACCENT title chip + DIM_BORDER) for
  the unfocused state and `focused_card_block` (GREEN title chip +
  GREEN border) for the card containing the cursor.
- Row marker `▌` indicates the selected item; `●` indicates marked
  items.
