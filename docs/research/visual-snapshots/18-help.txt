Notes:
- Search input at the top accepts characters typed while the overlay
  is open; cursor `▍` indicates focus.
- FAQ titles are bold; answers MUTED below.
- Shortcut rows use key chips + label. When a shortcut has a CLI
  equivalent (`Ctrl-p` → command palette etc.), an indented `CLI:` line
  appears below in MUTED.
- Filter matches against shortcut, label, FAQ question, and FAQ answer.
